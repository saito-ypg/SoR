## ゲーム概要


## ゲーム操作
エンター押すとplay開始

マウス右クリック：押した場所へ移動
Qキー：円形スキル

スキル発動概要
キーを押すと発動準備、そのまま発動。マウスの方向を向いて発動する。
準備中に右クリックをすると発動をキャンセルできる。
